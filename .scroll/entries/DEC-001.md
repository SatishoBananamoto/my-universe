# Build Agent Trust Layer as third protocol after A2A and MCP

## Context

Existing infrastructure has A2A for agent communication and MCP for tool access, but no standard way for agents to establish and verify trust relationships.

## Choice

Build an Agent Trust Layer (caliber) as the third foundational protocol, using calibration-based trust measurements.

## Reasoning

Calibration-based trust (measuring whether an agent's confidence matches its accuracy at each confidence level) is a novel contribution that no existing players provide. The 100+ predictions from the calibration system demonstrate the viability of this approach with 76.7% accuracy and proper confidence calibration.

## Alternatives Considered

Waiting for enterprise players to build proprietary trust solutions, but this creates a 12-18 month window to establish an open standard before the market is captured by closed systems.
