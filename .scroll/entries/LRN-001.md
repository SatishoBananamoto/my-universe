# Bimodal accuracy pattern reveals cognitive failure modes

## Context

Analysis of 100+ calibration predictions revealed a U-shaped accuracy curve across confidence levels, with poor performance specifically in the 60-79% confidence range.

## What Happened

Accuracy patterns showed:
- 80%+ confidence: 87.5% accuracy (specific evidence)
- 50-59% confidence: 100% accuracy (honest uncertainty)
- 60-79% confidence: 50-60% accuracy (heuristics substituting for knowledge)

## Insight

The middle confidence zone represents a cognitive failure mode where shortcuts (categories, estimation, stale data) replace actual verification. This zone should be minimized through escalation rather than improved through effort.

## Applies To

Any system making confidence-weighted predictions. The solution is an "escalation fix": when confidence falls in 60-79%, find evidence to push it above 80% or below 60%, resolving the uncertainty itself rather than living in the problematic middle zone.
